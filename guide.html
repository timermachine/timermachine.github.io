Fast Start Guide

I got frustrated by all the online timers out there not working properly or not having great features - so I made my own: Timer Machine!

Lets dive in! All the examples are ready to hit play. Guidance on the left.

----------------------------------------------------------------------------------------------

1. Countdown Timer

  1. Basic Countdown timer, set to play default alert sound after 5 seconds.
  2. Hover over buttons to learn what they do.
  3. Start it.
  4. Pause it.
  5. When paused, click Edit button to go back.
  6. Try some different sounds.
  7. Try some different times.

----------------------------------------------------------------------------------------------

2. A Talking Timer:

  1. Lets have some fun.This one will speak your text.
  2. Dont go pranking people ;)
  1. Note: in this sample: alarm sound [ ] alarm speak [√]
  2. Start the timer and hear it talk to you after 3 seconds.
  3. Go back to edit and type whatever you want it to say.
  4. Start it again.
  5. Try this: you can have both alarm sound and talking.
  6. Note: The talking plays once.

--------------------------------------------------------------------------------------------------------

3. Music/Video Playing Timers:

  1. Imagine having a timer that plays your choice of relaxing music as you go to sleep,
  2. and perhaps something more rousing when you wake. At no cost!
First lets get rid of Youtube Ads. Two options:
  • Paid for: Youtube Premium
  • 100% Free:Brave Browser
Recomendation: get Brave Browser, and use TimerMachine.com in it. Thats it! Works on Mobile and Desktop.

----------------------------------------------------------------------------------------------

3.1 Youtube links

  1. Ready? Lets play someting rousing when the timer is completed first:
  2. Click [v Advanced] on the Timer edit page.
  3. enable `on End Link`
  4. Copy any youtube page address. and paste it in.
  5. Start timer.
  6. Try the same but with On Start Playlist.

----------------------------------------------------------------------------------------------

3.2 Playlists

  1. Click [v Advanced] on the Timer edit page.
  2. Enable ‘on start playlist’
  3. Choose a playlist.
  4. Enable ‘on end playlist’
  5. Choose a playlist.
  6. Start timer.
  7. Note: when you have a start and end playlist,
  8. it will give a few seconds gap between (so you can wake up!)

----------------------------------------------------------------------------------------------

4. Intervals

  1. Intervals are useful for all sorts of timers like Meditation, Training.
  2. You can play sounds and or speak text - or both.
  1. Click ‘v Advanced‘ to open up more settings.
  2. Scroll to the lowest section- Interval.
  3. Set interval time. Note - this is by elapsed time.
  4. Start it. no need to close Advanced pannel
  5. Try interval with speach and or sound.

----------------------------------------------------------------------------------------------

5. Stopwatch

  1. No need for a separate app/buttons for a stopwatch.
  2. Just start a timer with no time set.
  3. Interval and Start Music settings will still work for stopwatches.
  4. hundreds of second resolution, laps coming soon.

6. Timer Library

  • the samples on this Guide page are all onetimer examples.
  • In the library you can keep multiple timers.
  • Try adding a timer.
  • Play it to save it.
  • refresh your browser. its still there
  • Note: it is saved in your browser. Great for privacy.
  • For sharing see the next section: Singe Timer. (Cloud saving coming soon)
